wipmap

Usage:
  wipmap --open
  wipmap --open --fullscreen
  wipmap --help
  wipmap --version

Options:
  -o, --open              Open browser.
  -p, --port=<PORT>       Run server on custom port (default is 8888).
  -f, --fullscreen        Open browser in kiosk mode.
  -h, --help              Show this screen.
  -v, --version           Print the current version.
